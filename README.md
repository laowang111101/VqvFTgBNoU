# Django图书管理系统 python916

本项目是一个基于Python语言的Django框架开发的图书管理系统，使用MySQL数据库进行数据存储，前端采用HTML、CSS、JavaScript以及Vue.js技术。

## 技术栈

- **前端：**
  - HTML
  - CSS
  - JavaScript
  - Vue.js
- **后端：**
  - Django框架
- **数据库：**
  - MySQL

## 功能模块

1. **用户登录功能**
2. **用户注册功能**
3. **管理员用户管理功能**
4. **管理员图书信息管理功能**
   - 管理图书名称、封面图片上传、简介等信息
5. **管理员图书状态查看**
   - 查看图书未被借走或已被借走状态
6. **管理员借阅信息审核**
7. **用户浏览图书信息**
   - 用户点击借阅按钮，完成图书借阅
8. **用户个人借阅信息管理**
   - 在此页面完成还书操作
9. **用户个人信息管理**
   - 更新个人信息

## 系统角色

- 普通用户：可以注册、登录、浏览图书信息、借阅和还书。
- 管理员：除了拥有普通用户的所有权限外，还可以管理用户信息、图书信息，以及审核借阅信息。
